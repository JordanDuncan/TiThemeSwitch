# TiThemeSwitch
Easily switch TiCh/Alloy themes.
## Installation
```
npm install -g tithemeswitch
```
## Usage
Run from withing the root of an Alloy project.

####Switch to theme
```
tithemeswitch (<themename> or <themeindex>)
```
####See installed themes with indexes
```
tithemeswitch -list
```
